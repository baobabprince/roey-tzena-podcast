# X-Radio Digest: Automated AI Podcast

An autonomous pipeline that transforms your Twitter (X) timeline into a professional, daily Hebrew radio digest. Powered by Gemini 2.0 and Google Cloud TTS.

## 🎙️ How it Works

1.  **Twitter Scraping**: Uses `twikit` to fetch the latest tweets from your home timeline. If the timeline is empty, it falls back to a curated list of top Hebrew news/tech accounts.
2.  **Engagement Ranking**: Filters out ads and ranks tweets by engagement (Likes + Retweets).
3.  **Deep Dive**: Fetches top replies for the most significant stories to provide context.
4.  **AI Script Writing**: Sends the curated data to **Gemini 2.0 Flash** with a specialized prompt to act as a sharp, witty radio editor.
5.  **Hebrew Neural TTS**: Converts the script into high-quality Hebrew audio using **Google Cloud Text-to-Speech** (`he-IL-Wavenet-A`).
6.  **RSS Distribution**: Updates an `rss.xml` feed and uploads the audio to GitHub Releases.

## 🛠️ Tech Stack

*   **Runtime**: Python 3.10
*   **Automation**: GitHub Actions (Scheduled runs at 07:00 and 17:00 IST)
*   **LLM**: Google Gemini 2.0 Flash
*   **Audio**: Google Cloud TTS (Neural/Wavenet)
*   **Scraper**: Twikit (Browser-based simulation)

## ⚙️ Configuration

The pipeline requires the following GitHub Secrets:

| Secret | Description |
| :--- | :--- |
| `X_USERNAME` | Twitter handle for scraping |
| `X_PASSWORD` | Twitter password |
| `X_EMAIL` | Associated email for login verification |
| `X_TOTP_SECRET` | 2FA Alphanumeric secret key |
| `GEMINI_API_KEY` | Google AI Studio API Key |
| `GCP_SA_KEY` | Google Cloud Service Account JSON (with TTS permissions) |
